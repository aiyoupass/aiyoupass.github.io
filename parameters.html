并行参数

参数名称

数据类型

说明

默认值

解释

BLOCK

int

设置每个进程单次计算分配到的任务量

1

所有任务被分成约BLOCK*NUMBER_OF_CORES份, 每个CPU核只计算其中至多BLOCK份

NUMBER_OF_CORES

int

可用核数

62

若不设置,默认为所有可用核数-1

NUMBER_OF_CORES_PC

int

PC模式下的核数

4

单独为PC端(家用CPU)设置CPU核数
